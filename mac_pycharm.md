## 常用快捷键  
* cmd + b 跳转到变量声明处  
* cmd + backspace 删除当前行  
* enter + shift  新建下一行
* enter + option + cmd  新建一行在光标上一行处  
* cmd + f  
* cmd + shift + f  
* cmd + r  
* cmd + shift + r  
* cmd + shift + -  折叠  
* cmd + shift + +  打开  
* cmd = /  注释与取消注释  
* cmd + D 复制代码到下一行  
* tbl 
* tbl + shift 缩进与反缩进
